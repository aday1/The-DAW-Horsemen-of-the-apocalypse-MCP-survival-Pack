Out of the box (streamlined)
----------------------------
  1. git clone https://github.com/aday1/The-DAW-Horsemen-of-the-apocalypse-MCP-survival-Pack.git DAW-Horsemen
  2. cd DAW-Horsemen
  3. CARE.bat
        OR double-click launch_daw_mcp.bat / Desktop "DAW Horsemen"
           -> GUI -> button "CARE — update + heal + SSE"
  4. Restart Cursor / Claude Desktop once
  5. Open DAWs (GUI Start buttons, or Desktop launcher)

CARE / GUI CARE does: git pull if behind, heal Bitwig/REAPER/Renoise + agent
MCP configs, free X-Touch for Bitwig, Desktop shortcut, shared SSE :8080,
health check.
