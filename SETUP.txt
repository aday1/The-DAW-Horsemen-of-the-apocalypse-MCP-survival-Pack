THE DAW HORSEMEN MCP PACK -- SETUP
==================================
GitHub: aday1/The-DAW-Horsemen-of-the-apocalypse-MCP-survival-Pack

Fresh machine
-------------
  git clone https://github.com/aday1/The-DAW-Horsemen-of-the-apocalypse-MCP-survival-Pack.git DAW-Horsemen
  cd DAW-Horsemen
  CARE.bat

CARE.bat = update-if-behind + heal + Mackie template + shortcut + health.
INSTALL.bat = deps + heal (still fine). UPDATE.bat = git pull + heal.

Release ZIP (maintainers):  RELEASE.bat  -> dist\ + GitHub release

Then read:  IDE_SETUP.txt   (Cursor, Claude Code, Claude CLI, VS Code, Desktop)


Day-to-day
----------
  Desktop shortcut OR  launch_daw_mcp.bat
    H = health check
    1 = Bitwig + shared MCP (SSE :8080)   <-- required for multi-agent Bitwig
    2 = REAPER + bridge
    3 = Renoise + ReMCP wake
    4 = all three

  Updates:  UPDATE.bat


What rides what
---------------
  REAPER   packages/reaper-mcp/            Python MCP + Lua bridge
  RENOISE  packages/renoise-mcp-bridge/    Node bridge -> ReMCP in Renoise
  BITWIG   packages/bitwig-mcp-server/     Shared SSE MCP + DawpocalypseMCP OSC
                                           (DrivenByMoss fork; recv 8005 / send 9001)
  REASON   packages/reason/                No own MCP - use REAPER + Rack Plugin


Bitwig shared server (important)
--------------------------------
  Run ONE: packages\bitwig-mcp-server\run_bitwig_mcp_shared.bat
  Every IDE uses: http://127.0.0.1:8080/sse
  Never spawn a second python -m bitwig_mcp_server for another IDE.
  Details: packages\bitwig-mcp-server\SHARED_SERVER.md
